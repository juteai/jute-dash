# Jute Mono

`jute-mono` is the default Jute Dash theme. It keeps the first visual system deliberately simple: black-on-white light mode, white-on-black dark mode, neutral borders, and semantic state accents only.

This folder is the first contribution shape for future repository-hosted themes. New themes should include a `theme.json`, document their intended use, and follow the rules in [Theme Guidelines](../../docs/developer/theme-guidelines.md).
